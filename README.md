# 視訊直播

## 作品說明
透過擷取本機影音流直播影像和聲音，使用者留言串討論功能，傳送表情符號。

使用Vue.js開發，使用navigator.mediaDevices取得攝影機影音。

Demo：[https://live-stream-demo.netlify.app/](https://live-stream-demo.netlify.app/)

## 使用技術
* Vue.js
* SCSS
* navigator.mediaDevices
